Service: Supabase
Definition:
Supabase is an open-source Firebase alternative built on PostgreSQL, providing a scalable backend with real-time data, authentication, and storage features.
Implementation:

Stores structured hospital data (patients, appointments, claims, staff).

Uses PostgreSQL under the hood with Supabase client integration in Django and React.

Real-time synchronization for live updates in patient queues and laboratory results.

Implements Row Level Security (RLS) for patient confidentiality and access control.
