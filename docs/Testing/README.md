# WAH4Hospital | Testing & Quality Assurance

This directory contains the complete testing documentation and audit trail for the **WAH4Hospital** system. It serves as a centralized hub for all information regarding system analysis, test execution, and results.

## Directory Structure

* **[`evidence/`](./evidence/)**: Contains visual proof (screenshots) of test executions, organized by testing dates and system versions.
* **[`test-cases/`](./test-cases/)**: Includes detailed test scenarios and the master link to our SharePoint Test Case ledger.

## Purpose
The goal of this documentation is to maintain full traceability between the software development commits and the manual verification performed by the testing team.
